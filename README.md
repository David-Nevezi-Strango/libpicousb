# libpicousb
Personal project developing a low level, distributed application involving USB communication between ARM microcontroller and Host Linux PC


# Learning material
- Miro Samek's Modern Embedded Systems Programming. (Not followed to every detail as the Pico 2 is more advanced than TI/STM32)
- Pico C/C++ SDK's examples and the SDK itself.

Additionally, LLMs were utilized as guidance for learning, understanding concepts and debug purposes.
